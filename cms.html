EXPORT — MAKE CHANGES PERMANENT

Why export? CMS changes are stored in your browser. If you open the site on another device or clear browser data, changes are lost. Exporting creates a new data.js with your edits baked in — upload it to GitHub to make everything permanent everywhere.

1. Click Export data.js
2. Download the file
3. On GitHub, go to assets/js/data.js → click the pencil icon → paste the content → commit
4. Done — changes are now permanent on all devices!
